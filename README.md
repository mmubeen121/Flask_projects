# Flask Projects

This repository contains a collection of **Flask projects** built to practice and master **Python backend development using Flask**. The goal of this repository is to gradually build real-world applications while learning core backend concepts such as routing, templating, databases, authentication, and API development.

Each project in this repository focuses on different aspects of Flask and web development, starting from basic applications and progressing toward more advanced and production-style projects.

---

## Purpose of the Repository

The main purpose of this repository is to:

* Practice **Flask backend development**
* Learn how to structure Flask applications
* Implement **Object-Oriented Programming (OOP)** in Flask
* Work with **HTML, CSS, and JavaScript** for frontend integration
* Build practical projects using **SQLAlchemy databases**
* Develop real-world backend skills step by step

This repository will grow over time as more projects are added.

## How to Use

1. Clone the repository

```
git clone https://github.com/yourusername/Flask_projects.git
```

2. Navigate into a project directory

```
cd project-folder
```

3. Install dependencies

```
pip install flask
```

4. Run the application

```
python app.py
```
