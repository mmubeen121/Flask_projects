# Flask Portfolio Website (Practice Project)

This repository contains a **personal portfolio website project built for practicing Flask development**.
The main goal of this project is to learn how to structure a backend application using **Python, Flask, and Object-Oriented Programming (OOP)** while integrating **HTML, CSS, and JavaScript for the frontend**.

The project is currently a **learning project**, and the repository is being used to practice building and pushing Flask-based applications to GitHub.

---

## Project Goal

The goal of this project is to practice and understand:

- Flask application setup
- Object-Oriented Programming in Flask
- Backend routing
- Integrating frontend pages with Flask
- Structuring a backend-driven portfolio website

---

## Technologies Used

- **Python**
- **Flask**
- **HTML**
- **CSS**
- **JavaScript**
- **Object-Oriented Programming (OOP)**

## Features

- Flask application structured using a class-based approach
- Basic route handling
- Static frontend pages created with HTML, CSS, and JavaScript
- Minimal and professional portfolio layout
- Backend-ready architecture for future expansion
