# Hello Web App

Hello Web App is a simple beginner-level web application built using Flask.  
The purpose of this project is to understand the basic structure of a Flask application, including routing and running a web server.

## Project Overview

This project demonstrates how to create a minimal Flask web application that serves a single route and returns a response in the browser.

When the server runs and you open the root URL, the application displays the message:

Welcome to Hello Web App

## Features

- Simple Flask application structure
- Class-based Flask app implementation
- Basic route handling
- Debug mode enabled for development

## Technologies Used

- Python
- Flask

## Run in Terminal

- use "python app.py" cmd
